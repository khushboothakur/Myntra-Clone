## ✨ Features
 
- 🧭 **Sticky Navbar** — Search bar, cart badge, wishlist & profile
- 🗂️ **Category Navigation** — MEN, WOMEN, KIDS, BEAUTY, SALE & more
- 🎯 **Hero Banner** — Animated gradient with CTA buttons
- 🛍️ **Shop by Category** — 6 category cards with hover effects
- 🔥 **Hot Deals** — 3 themed offer cards
- 👕 **Product Grid** — 8 cards with brand, price, rating & discount
- ❤️ **Wishlist Toggle** — Add/remove from wishlist
- 🛒 **Cart Slide Panel** — Add to bag, remove, view total
- 🔔 **Toast Notifications** — Animated feedback on actions
- 🏃 **Brand Marquee** — Auto-scrolling brand logos
- 📱 **Fully Responsive** — Mobile, tablet & desktop

## 🚀 Getting Started
 
### Clone karo
```bash
git clone https://github.com/khshboothakur/Myntra-Clone.git
cd Myntra-Clone
```
 
### Run karo
```
index.html ko browser mein open karo — no setup needed!
```
 
> 💡 **Tip:** VS Code mein **Live Server** extension use karo for auto-refresh.
