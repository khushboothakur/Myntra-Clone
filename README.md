# 🛍️ Myntra Clone

A fully responsive **Myntra-inspired fashion e-commerce website** built using pure **HTML, CSS, and JavaScript** — no frameworks, no libraries.

## 🌐 Live Demo

> Open `index.html` directly in your browser — no server needed!


## 📸 Features

- 🧭 **Sticky Navbar** — with search bar, cart badge, wishlist & profile icons
- 🗂️ **Category Navigation** — MEN, WOMEN, KIDS, BEAUTY, SALE & more
- 🎯 **Hero Banner** — animated gradient with CTA buttons and dot indicators
- 🛍️ **Shop by Category** — 6 category cards with hover effects
- 🔥 **Hot Deals Section** — 3 offer cards (Pink, Dark, Gold themes)
- 👕 **Product Grid** — 8 product cards with brand, price, rating & discount
- ❤️ **Wishlist Toggle** — add/remove products from wishlist
- 🛒 **Cart Slide Panel** — add to bag, remove items, view total
- 🤍 **Wishlist Slide Panel** — move items from wishlist to bag
- 🏷️ **Filter Chips** — Top Rated, New Arrivals, Best Sellers, etc.
- 🏃 **Brand Marquee Strip** — auto-scrolling brand logos
- 🔔 **Toast Notifications** — animated feedback on every action
- 🦶 **Footer** — links, social icons, copyright
- 📱 **Responsive Design** — works on mobile, tablet & desktop
